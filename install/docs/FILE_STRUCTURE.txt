USAGE PATHS
===========

Installation:
  ~/hyprconfig/install.sh
  → Prompts for fingerprint
  → Installs packages
  → Runs fingerprint.sh
  → Enrolls fingerprints

Quick Setup:
  ~/hyprconfig/setup-fingerprint.sh
  → Checks installation
  → Runs full setup
  → Ready to use

Management:
  ~/hyprconfig/install/fingerprint-manager.sh [command]
  
  Commands:
    enroll [finger] .... Enroll new fingerprint
    list ............... Show enrolled fingerprints
    delete [finger] .... Remove specific fingerprint
    delete-all ......... Remove all fingerprints
    test ............... Test authentication
    status ............. Show service status
    restart ............ Restart fprintd service
    help ............... Show help message

Documentation:
  ~/hyprconfig/install/docs/FINGERPRINT_SETUP_REPORT.md
  ~/hyprconfig/install/docs/FINGERPRINT_CHANGES.txt (this file)
